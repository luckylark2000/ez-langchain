# 安装 python 环境

## 编辑器

编辑器使用的是 vscode

安装对应vscode里面的 python 的插件，推荐插件市场直接搜索 `python extension pack` 然后就安装里面几个没有 deprecated 的包基本就可以了

类似于 `javascript` 、`vue extension pack` 、`react extension pack`

## python 安装和版本管理

我这里直接使用的 uv，类似于前端的 nvm 工具，但是更加强大

详细地可以参考我的，**uv 核心知识学习** 这篇文章

不但能够在项目中自动使用对应版本的 python，还能够执行一些类似 npm, 安装包的功能，创建python 项目环境

可以非常快速地进行python项目的搭建
