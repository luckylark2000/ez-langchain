## 变量

变量声明不用用类似 let, const 之类的关键字，python 自动识别

从python 3.9版本开始类型声明，泛型之类的，编辑器中如果想要有类型错误提示需要安装 pyright vscode 插件

```py
# 变量声明
num = 3 

## 带类型的（3.9版本开始支持）,类型提示需要安装pyright插件
a:bool = True

print(num) # 3
print(a) # True
```
